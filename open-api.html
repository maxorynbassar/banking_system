Open API және Open Banking Қазақстанда

Open API (Ашық бағдарламалау интерфейсі) және Open Banking тұжырымдамасы банктерге өздерінің деректері мен қызметтерін үшінші тарап әзірлеушілеріне қауіпсіз түрде ұсынуға мүмкіндік береді. Бұл инновациялық қаржылық өнімдер мен қызметтерді құруға ықпал етеді.

Негізгі мақсаттар (2.2-бөлімнен):

  • Қаржы нарығындағы бәсекелестікті дамыту.
  • Инновацияларды ынталандыру.
  • Клиенттер үшін қызметтердің сапасы мен қолжетімділігін арттыру.
  • Қауіпсіз ортаны қамтамасыз ету.

Open API экожүйесінің қатысушылары (Кесте 16-дан):

№ Қатысушы Рөлі Функциясы
1 ҚРҰБ, ҚНРДА, БҚДА Реттеушілер Жобаның дамуын бақылау
2 Қаржы ұйымдары API жеткізуші Ашық API ұсыну

Пилоттық жобалар:

Қазақстанда бірнеше пилоттық жобалар іске асырылды, соның ішінде:

  • Клиент шоттарын біріктіру: Бір қосымшада әртүрлі банктердегі шоттар туралы ақпаратты көру.
  • Банкаралық QR-төлемдер: Бірыңғай QR-код арқылы төлем жасау.

// GET /accounts/{user_id}/balance
// Host: bank-api.kz
// Authorization: Bearer {token}
